# 👟 Nike Sales Analysis

## 📌 Project Overview
This project focuses on analyzing Nike sales data to uncover key business insights related to sales performance, customer segments, and product trends. The analysis was performed using Power BI to create an interactive and visually appealing dashboard.

---

## 🎯 Objectives
- Analyze overall sales and profit performance  
- Identify top-performing products and categories  
- Understand customer segmentation (Gender-wise analysis)  
- Evaluate regional sales distribution  
- Track monthly and yearly sales trends  

---

## 📊 Dataset Description
The dataset contains the following columns:

- Order ID  
- Order Date  
- Product Name  
- Product Line  
- Category  
- Gender  
- Size  
- MRP (Sales Amount)  
- Profit  
- Region  
- Sales Channel  

---

## 🛠️ Tools & Technologies Used
- Power BI (Data Visualization & Dashboard)  
- DAX (Data Analysis Expressions)  
- Power Query (Data Cleaning & Transformation)  

---

## 🔍 Key Analysis Performed
- Created KPIs for Total Sales, Total Profit, and Total Orders  
- Performed **year-wise monthly trend analysis**  
- Analyzed **sales by region and category**  
- Compared **sales across gender segments**  
- Evaluated **sales channel performance (Online vs Offline)**  
- Identified **top-performing products and categories**  

---

## 📈 Key Insights
- Certain regions contribute significantly to overall sales  
- Seasonal trends observed in monthly sales patterns  
- Specific product categories generate higher profit margins  
- Customer preferences vary across gender and product lines  
- Sales channels show different performance levels  

---

## 📊 Dashboard Features
- Interactive filters (Region, Gender, Category)  
- Dynamic charts (Line, Bar, Pie)  
- KPI cards for quick insights  
- User-friendly layout for easy navigation
